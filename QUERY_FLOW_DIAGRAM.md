## Key Components & Their Roles

| Component | File | Purpose |
|-----------|------|---------|
| **Frontend** | `script.js` | User interface, sends HTTP requests |
| **API Layer** | `app.py` | FastAPI endpoints, request/response handling |
| **RAG System** | `rag_system.py` | Main orchestrator, coordinates all components |
| **Session Manager** | `session_manager.py` | Maintains conversation history |
| **AI Generator** | `ai_generator.py` | Interfaces with Claude API, handles tool calling |
| **Tool Manager** | `search_tools.py` | Manages and executes tools |
| **Course Search Tool** | `search_tools.py` | Formats search requests and results |
| **Vector Store** | `vector_store.py` | ChromaDB interface for semantic search |

## Data Flow Summary

1. **User Input** â†’ Frontend captures query
2. **HTTP POST** â†’ Backend API receives request
3. **Orchestration** â†’ RAG System coordinates processing
4. **Context Retrieval** â†’ Session Manager provides history
5. **AI Processing** â†’ Claude receives query with tools
6. **Tool Decision** â†’ Claude decides to search
7. **Tool Execution** â†’ Search tool is invoked
8. **Semantic Search** â†’ Vector store queries ChromaDB
9. **Result Formatting** â†’ Search results formatted with sources
10. **AI Synthesis** â†’ Claude generates final answer
11. **Response Assembly** â†’ Answer + sources combined
12. **HTTP Response** â†’ JSON returned to frontend
13. **UI Update** â†’ Chat interface displays result
